Duke

Project details

The Duke chatbot is a Command Line Interface (CLI) application developed individually using the JAVA programming language. Duke allows users to record down and manage tasks.

Features

  • Adding Tasks

    • Adds tasks like To-dos, Events and Deadlines.
    To-dos are tasks to be done, and do not have a deadline.
    Events are "tasks" with a start and end date.
    Deadlines are tasks with a deadline.

  • Checking Tasks

    • Marks a task in your list as completed.

  • Unchecking Tasks

    • Marks a task in your list as incomplete.

  • Searching Task List

    • Allows searching of task list by name

  • List

    • Displays the task list.

  • Deletes an item from the task list

    • Deletes a task from the task list.
