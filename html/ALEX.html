Alex to the Rescue

Gallery


Project details

Alex is a robot built on the Robot Operating System. Alex is used to navigate a maze remotely, using a LiDAR system to map out the maze, visualised using hector_slam. Alex is built in a team of 4.

Personal Contributions

Wrote the software behind Alex in Arduino Bare-Metal programming and calibrated wheel movement for the "WASD" control scheme in C.
